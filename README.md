# Super Bot

Super Bot is a simple Slack bot built using the Slack Bolt framework. 

## Features

- `/super_bot-help` – Shows all available commands.
- `/super_bot-catfact` – Returns a random cat fact.
- `/super_bot-joke` – Tells a random joke.
- `/super_bot-quote` – Shares an inspirational quote.
- `/super_bot-dadjoke` – Tells a dad joke.
- `/super_bot-8ball` – Answers your question like a Magic 8-Ball.
- `/super_bot-coinflip` – Flips a coin.
- `/super_bot-rps` – Play Rock, Paper, Scissors with the bot.

## Technologies

- Node.js
- Slack Bolt
- JavaScript
